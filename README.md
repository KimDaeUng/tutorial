# MLOps for MLE Tutorial

"머신러닝 엔지니어를 위한 MLOps" 튜토리얼 문서가 있는 레포입니다.

## How to start
### Install npm

이 레포는 [Docusaurus 2](https://docusaurus.io/) 로 build 되며 백엔드는 npm을 사용합니다.

- 요구사항
    - `Node.js>=16.14`


### Check local build

수정한 후 로컬에서 실행이 되는지 확인합니다.

```bash
npm run start
```

### pre-commit
pre-commit 을 통과하는지 확인합니다.

```bash
pre-commit run --all-files
```
